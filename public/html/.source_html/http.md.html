Message Types 报文格式

  • Start Line
  • Message Headers
  • Message body

Request 请求

  • Request Line 请求行
  • General Header 通用头
  • Request Header 请求头
  • Entity Header 实体头
  • Message Body 报文主体

Response 响应

  • Status Line 状态行
  • General Header 通用信息头
  • Response Header 响应头
  • Entity Header 实体头
  • Message Body 报文主体

GET 和 POST 区别

  • GET >> 数据 >> Request Line 中的 URL
  • POST >> 数据 >> Message Body

参考文献

https://tools.ietf.org/html/rfc7230
